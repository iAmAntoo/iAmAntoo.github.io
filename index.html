Features

API Endpoints / Features

Cosmetics

Data of all in-game cosmetics including leaked/unreleased items

Item Shop

Data of the daily & featured item shop which gets automatically updated at 0:00 AM UTC

Player Stats

Data of battle royale player stats including with optional image generation (example)

News

Data of battle royale, save the world & creative news with automatically generated GIFs (example)

Playlists

Data of all playlists (gamemodes)

Minimap

Data of the current minimap with POIs and automatically generated POI images (example)

Banners

Data of all profile banners

AES Keys

Data of all AES keys which can be used to decrypt the game's pak files

Creator Code

Data of a creator code such as the status and info of its account
